# SW-5.1 1000W Equalizer APO Configuration

This Equalizer APO configuration is designed for connecting the SW-5.1 1000W sound system directly to a low-end Realtek sound card.

I created this configuration because the bass channel was not being properly routed to the subwoofer, resulting in weak bass response and poor overall audio quality. This setup improves bass redirection and helps restore a more balanced 5.1 audio experience.

## Important Note for Sound Blaster Users

**Do not use this configuration with Sound Blaster sound cards.**
Instead, enable **Bass Redirection** through the Sound Blaster control software, as it provides the proper routing and processing for those devices.

## Requirements

* Equalizer APO installed
* A Genius SW-5.1 1000W

## Notes

This configuration was made specifically for systems using basic Realtek audio hardware and may not be suitable for all sound cards or speaker setups. Results may vary depending on your hardware and drivers.
